# Audio offline (reproductor)

1. Coloca aquí archivos de audio (por ejemplo `.mp3`, `.ogg`).
2. Registra cada pista en `src/app/modules/shared/music-player/offline-music-playlist.ts` con `title` y `src` relativo, p. ej. `assets/audio/mi-tema.mp3`.
3. Usa solo música con derechos claros para redistribuir en la app.
